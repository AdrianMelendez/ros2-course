## Quick Start
```bash
docker compose build
docker compose up -d
docker exec -it ros2-humble bash

cd /home/adml/ros2_ws
colcon build --symlink-install
source install/setup.bash

docker compose down
```

This setup mounts the current directory into `/home/adml/ros2_ws` so you can build existing repo files in place.
